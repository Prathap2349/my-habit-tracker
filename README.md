# my-habit-tracker

A beautiful and responsive weekly habit tracker dashboard built using pure HTML, CSS, and JavaScript.

## ✨ Features

- ✅ Weekly task management
- 📅 Daily progress tracking
- 🔥 Streak counter
- 😊 Mood tracker
- 📊 Habit tracker with completion percentage
- 🎨 Custom themes
- 🌙 Dark mode
- 🎉 Confetti animations
- 📸 Export weekly summary as image
- 💾 Local storage support
- 📱 Fully responsive design
